# CampusRaid

Це мобільний додаток для хакатону. Він написаний на **React Native (Expo)**. 

Щоб судді та інші користувачі могли **безкоштовно** та миттєво встановити цей додаток на свої iPhone та Android (без необхідності завантажувати Expo Go або створювати акаунт розробника Apple), ми використовуємо технологію **PWA (Progressive Web App)**.

---

## 📱 Як встановити додаток (Для Користувачів та Суддів)

Вам **не потрібно нічого компілювати або налаштовувати**. Просто відкрийте посилання, яке надав вам розробник!

### 🍎 Для iPhone (iOS):
1. Відкрийте посилання у стандартному браузері **Safari**.
2. Натисніть кнопку **Поділитися** (квадратик зі стрілочкою внизу по центру).
3. Прокрутіть меню вниз і оберіть **На початковий екран** (Add to Home Screen).
4. Натисніть **Додати**.
*✅ Додаток з'явиться на робочому столі. При відкритті він працюватиме на весь екран, як справжній нативний додаток!*

### 🤖 Для Android:
1. Відкрийте посилання у браузері **Chrome**.
2. Внизу екрану автоматично з'явиться банер **"Додати CampusRaid на головний екран"** (або можна натиснути 3 крапки в правому верхньому куті -> **Додати на гол. екран**).
3. Підтвердіть встановлення.
*✅ Додаток встановиться і з'явиться в меню всіх додатків.*

## 🚀 Як задеплоїти оновлення (ТІЛЬКИ ДЛЯ РОЗРОБНИКІВ)

Якщо ви розробник і хочете задеплоїти свій код на Vercel:
1. Зайдіть на [Vercel.com](https://vercel.com/) та авторизуйтесь через GitHub.
2. Натисніть **Add New... -> Project**.
3. Знайдіть репозиторій `CampusRaidFront` і натисніть **Import**.
4. У налаштуваннях проекту (Build and Output Settings):
   - **Framework Preset**: залиште `Other`
   - **Build Command**: `bun run build` (або `npm run build`)
   - **Output Directory**: `dist`
5. Натисніть **Deploy**.
