# 📱 CampusRaid Frontend (Expo + NativeWind + Zustand)

CampusRaid — це мобільний додаток для гейміфікованої освітня платформи, де вивчення програмування перетворюється на колективні битви з епічними ІТ-монстрами (Босами).

Фронтенд побудований за принципом **«кросплатформно, швидко, з гарним UI»**. Замість класичного React ми використовуємо **Expo (React Native)** для нативної роботи на iOS та Android, а для стилів — **NativeWind** (TailwindCSS).

---

## 🛠️ Технологічний Стек

- **Фреймворк**: [Expo](https://expo.dev/) (React Native) SDK 54
- **Стилізація**: NativeWind (TailwindCSS)
- **Стейт-менеджмент**: Zustand
- **Навігація**: React Navigation v7 (Bottom Tabs, Native Stack)
- **Мережеві запити**: Axios
- **Збереження даних**: AsyncStorage

---

## 🏗️ Швидкий Старт

1. **Встановлення залежностей:**
   Переконайтеся, що ви знаходитесь у папці `front`, і виконайте:
   ```bash
   bun install
   ```

2. **Запуск сервера (в режимі розробки):**
   Для локального тестування (на одному Wi-Fi):
   ```bash
   bun start -c
   ```
   *Відскануйте QR-код з терміналу за допомогою камери вашого телефону (iOS) або безпосередньо в додатку Expo Go (Android/iOS).*

3. **Спільний доступ (Тунель через інтернет):**
   Якщо вам треба показати додаток комусь поза вашою мережею:
   ```bash
   ssh -R 80:localhost:8081 nokey@localhost.run
   ```
   *Скопіюйте посилання `https://...`, замініть `https` на `exp` і відправте друзям.*

## 🛣️ Специфікація Екранів (Навігація)

Додаток складається з кількох головних екранів, доступних через нижню панель навігації (Bottom Tabs), та додаткових модальних вікон.

### 🔑 Блок 1: Авторизація
- **`LoginScreen`**
  - *Опис:* Єдиний екран для входу та реєстрації. Користувач вводить Email, Пароль, Унікальний Нікнейм та Університет.
  - *Особливості:* Токен зберігається в `AsyncStorage`, а стан юзера тримається в `useAuthStore` (Zustand).

### 👾 Блок 2: Ігрова логіка (Рейд на Боса)
- **`BossRaidScreen`**
  - *Опис:* Екран поточної битви з масивним босом.
  - *Механіка:* Залежно від типу завдання (SORTING чи MULTIPLE_CHOICE), гравець або складає правильну послідовність блоків коду, або обирає правильну відповідь A/B/C/D.
  - *Особливості:* При успішній атаці (`Execute Attack`) шкала HP боса зменшується, а гравець отримує XP та монети.

### 🏆 Блок 3: Лідерборд (Рейтинги)
- **`LeaderboardScreen`**
  - *Опис:* Топ-10 найкращих студентів на платформі за кількістю XP. Також присутній перемикач на рейтинг Університетів (по сумарному XP їхніх студентів).
  - *Особливості:* Динамічне завантаження списку.

### 💬 Блок 4: Приватні Чати
- **`MessagesScreen`**
  - *Опис:* Список ваших активних діалогів з іншими користувачами.
- **`ChatRoomScreen`**
  - *Опис:* Приватна кімната переписки між вами та конкретним юзером.
  - *Особливості:* Відображення історії повідомлень (бульбашки чату) з миттєвим оновленням.

### 🛒 Блок 5: Магазин (Shop)
- **`ShopScreen`**
  - *Опис:* Тут гравці можуть витрачати зароблені монети на ігрові предмети (наприклад, унікальні титули, рамки, підказки для босів).
  - *Особливості:* Перевірка балансу юзера перед покупкою.

### 📰 Блок 6: Форум (Reddit-style)
- **`ForumScreen`**
  - *Опис:* Стрічка публікацій вашого університету.
  - *Механіка:* Можливість публікувати меми, ставити запитання, а також система "Upvote/Downvote" для підняття кращих постів у топ.

## 🚀 Деплой та Збірка (EAS)
1. **Збірка APK/AAB для Android:**
   ```bash
   eas build -p android --profile preview
   ```
2. **Збірка для iOS (через TestFlight):**
   ```bash
   eas build -p ios --profile production
   ```
3. *(Для роботи EAS потрібен налаштований акаунт Expo та файл `eas.json`).*
