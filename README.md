# 🎮 CampusRaid — Frontend

**CampusRaid** — це гейміфікована освітня PWA-платформа для студентів та викладачів, де вивчення програмування перетворюється на епічні битви з IT-монстрами (босами). Написаний на **React Native + Expo**, розгорнутий як **Progressive Web App** через **Vercel**.

---

## 📱 Як встановити додаток (Для Користувачів та Суддів)

Вам **не потрібно нічого компілювати або налаштовувати**. Просто відкрийте посилання, яке надав вам розробник!

### 🍎 iPhone (iOS / Safari):
1. Відкрийте посилання у **Safari**.
2. Натисніть кнопку **Поділитися** (📤 внизу по центру).
3. Оберіть **На початковий екран** (Add to Home Screen).
4. Натисніть **Додати**.

> ✅ Додаток з'явиться на робочому столі і працюватиме на весь екран, як справжній нативний додаток!

### 🤖 Android (Chrome):
1. Відкрийте посилання у **Chrome**.
2. Натисніть 3 крапки → **Додати на головний екран**.
3. Підтвердіть встановлення.

> ✅ Додаток встановиться і з'явиться в меню всіх додатків.

## 🔐 Аутентифікація та Стейт

Після логіну сервер повертає `{ token, user }`. Токен і дані юзера зберігаються у Zustand з увімкненою персистентністю (через `AsyncStorage`). При наступному відкритті додатку — сесія відновлюється автоматично без повторного входу.

**API клієнт** (`src/api/client.ts`) автоматично підставляє `Authorization: Bearer <token>` до кожного запиту через Axios-інтерцептор.

## 🎮 Ключові Features

### 👾 Boss Raid (Головний геймплей)
- Отримай активного боса для свого університету
- Вирішуй пазли двох типів: **Multiple Choice** та **Code Ordering**
- При успіху: анімація тряски боса, нарахування XP та монет
- При помилці: красивий модальний попап з деталями

### 📰 Campus Forum
- Reddit-стиль пости з upvote/downvote
- Сортування за рейтингом (upvotes − downvotes)
- Створення постів через красивий centered-модал з glassmorphism
- Pull-to-refresh для оновлення стрічки
- Натисни на автора → надішли запит у друзі або відкрий чат

### 💬 Приватний Чат
- Real-time оновлення через polling (кожні 3 секунди)
- Бейдж непрочитаних повідомлень на іконці "Friends"
- Розпізнавання власних та чужих повідомлень

### 🏆 Лідерборд
- Топ студентів по XP
- Рейтинг університетів по сумарному XP

### 🛍️ Магазин VIP-Титулів
- Купуй за монети (які заробляєш у боса)
- Екіпіруй титул → він відображається на форумі з анімацією

### 🎵 Звукова Система
- Фонова музика з захистом від подвійного запуску (lock-механізм)
- SFX при виборі відповіді та критичному ударі
- Тактильна вібрація (haptics) при кожній дії
- Перемикач звуку в профілі

## 🚀 Деплой на Vercel

### Автоматичний (рекомендований)
Кожен `git push` в `main` → Vercel автоматично збирає та деплоїть.

### Ручний перший деплой
1. Зайдіть на [vercel.com](https://vercel.com/) та авторизуйтесь через GitHub.
2. **Add New → Project** → знайдіть `CampusRaidFront` → **Import**.
3. Налаштування білду:
   - **Framework Preset**: `Other`
   - **Build Command**: `bun run build`
   - **Output Directory**: `dist`
4. **Deploy**.

### Що робить `bun run build`?
```bash
expo export -p web   # Компілює React Native → статичні HTML/JS/CSS файли
node fix-html.js     # Додає PWA маніфест, тег viewport, apple-touch-icon
```
